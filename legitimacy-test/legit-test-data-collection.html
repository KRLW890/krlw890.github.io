I'm making a Pokemon Legitimacy Test, and I need help gathering information.

Even with the vast collection of pre-made Pokemon libraries available on the web, I still need to gather some information by hand, and I need a LOT of information.
Right now, I'm working on making a library I can use to determine which Pokemon are legal in which Pokeballs. Here's how it works:

I've made a simple program for easy data gathering, available here. When opened, it will ask you what generation you want to start from, and then lead you to a screen that looks something like this:

Here's what I need:

  • Unless otherwise specified, please exclude all events and NPC link trades. They come with additional information that this library doesn't need to address.
  • "Wild" should be true if the Pokemon can be fought and captured normally in the wild in ANY of the games indicated by the column. What you're looking for here is whether the Pokemon can be captured by any Pokeballs regularly available in those games. As such, this includes stationary and roming legendaries, but excludes Pokemon only catchable in the Safari Zone or Johto's Bug-Catching Contest. This should also include Pokemon that evolve from Pokemon who meet this qualification, since they can legally have these Pokeballs in the indicated games without trading. Even if an event is required to battle the legendary (ie, Eon Ticket event with Mew), as long as they can be caught with any Pokeball regularly obtainable, they meet this qualification.
  • "Received" should be true if the Pokemon can't be found normally in the wild, but can be obtained as gifts to the player in a standard Pokeball. ie, the starters, Game Corner prizes, etc.
  • "Safari" should be true if the Pokemon can be caught in the Safari Zone or Great Marsh.
  • "Sport" should be true if the Pokemon can be caught in Johto's Bug-Catching Contest. GSC and HGSS only.
  • "Dream" should be true if the Pokemon can be caught in the Dream World. BW/B2W2 only.

The data gathering program is designed to work with one generation at a time. When done, the program will construct a file with the information you inputted. Download this file and email it to me at krlw890@outlook.com.
