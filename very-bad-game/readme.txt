The main instructions are available in-game.  This document is mainly to explain tips and
tricks (although I would definitely recommend reading instructions!).

    **CONTENTS:
The Title Here zip file should contain several files: a file folder named "data", this
readme file, and the actual game, which, when launched, will open up in a web browser.
This does not mean you need internet connection to open or play the game.  If you move
the game file, move the entire "Title Here" folder, not just the single file.  Outside
of that, do not move or rename the "data" folder or any of its contents.


    **LAUNCHING:
As stated above, when launched, Title Here will open in a web browser.  **Title Here was
made for Google Chrome. If you use another web browser, it might not be able to run, but
if it does, the text will be out of place.  Gameplay may or may not be affected by using
another browser, but best case scenario, it will run as intended with misplaced text. **

tl;dr: For best results, use Google Chrome.

The Title Here file should have the icon of a browser.  This will be the browser the game
will default to opening with.  If it is not Google Chrome, right-click on the file and
select "Properties".  There should be a category called "Opens with:" along with the
browser.  Select "Change..." and then click on Chrome.


    **GLITCHES:
If you get to a point where you need to type, you might find that backspace does not work.
This is the fault of Chrome (seriously, you need to launch this with Chrome), but this
extension should fix it:
https://chrome.google.com/webstore/detail/backstop/pidcjgldchekcoolelhbjfbnccjkckfj?hl=en-GB
If you are still having trouble, the delete key is programmed to work as a replacement
for backspace.


    **TRICKS:
If you make a shortcut for the game, you can set it so that it shows the Title Here icon.
Right-click on the shortcut, open "Properties", then select "Change Icon...".  Select
"Browse...", then find the Title Here folder in the file explorer.  Open the folder
called "data", then select "icon" (It should be the only object available).

After completing the game at least once, you will be able to activate certain tricks.
Open up the browser's console (right click + "Inspect". Select the "Console" subtab if
not already selected) then type in "neatTricks" to get a list of cheat codes.  Keep in
mind that all codes are case and symbol-sensitive, except for the quotations.
